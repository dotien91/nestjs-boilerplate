# TFT Icons

Icons được tải từ [MetaTFT.com](https://www.metatft.com/icons)

## Cách tải icons

Chạy script:
```bash
node scripts/download-icons.js
```

## Sử dụng

Icons có thể được sử dụng trong frontend hoặc hiển thị trong API responses.

Ví dụ:
```html
<img src="/icons/AS.svg" alt="Attack Speed" />
```
